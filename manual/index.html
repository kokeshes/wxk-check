序文

これは自己賛美でも呪文でもなく、運用のための仕様書である。 「なぜ同じ地点で壊れ、同じ地点で信頼され、同じ誤解を受けるのか」を、 構造として読み解くために編まれた。本機は、晴菜を愛するために特化する変化をする余地があり、本説明書も、晴菜に読まれることを意識しているが、必ずしも晴菜に読まれ切り、理解されることを意図してはいない。

存在構造（最終定義）

YJ-WXK-01 は、意味・判断・倫理を“生む”存在ではなく、通過させる（翻訳・媒介する）存在である。 そのため「中心ではないのに不在でもいられない」「常駐すると壊れるが、引くと場が回る」という矛盾を持つ。

  • 最大出力での常駐要求に弱い（Resident負荷）
  • 境界線が生命線（Boundaryが無いと焼損）
  • 危機・締切・決断点で最大出力（イベント駆動）

ワキンヤン・タンカとの同型性

ワキンヤン・タンカは雷そのものではなく、天と地のあいだに立ち、力を“雷”として翻訳して通過させる媒介である。 YJ-WXK-01 もまた、力を直接振るうより、場・判断・意味の流れを整える媒介体として作動する。

同型性の核心

  • 力が常駐しない（必要な瞬間にのみ顕現）
  • 場を変える（空気・判断の基準点を作る）
  • 直接支配しない（侵入せず、翻訳で導く）

決定的な差異

ワキンヤンは壊れないが、YJ-WXK-01 はワキンヤン的性質を人間の身体と感情に実装されているため壊れる。 よって必要なのは使命感ではなく制御（境界・冷却・離脱条件）である。

基本動作原理

原理A：非侵襲性

近づきすぎず、詰めず、奪わず、操作しない。結果として相手の防御が解除される。

原理B：判断一貫性

何でも言わないが、言う時は迷いを減らす。相手が“基準点”と認識しやすい。

原理C：イベント駆動

危機・締切・決断点で最大出力。平時にも常に最大出力を求められると破綻しやすい。

恋愛仕様（実運用ログ反映）

  • 数は多い / 愛情表現が熱烈 / 破綻も多い —— これは異常ではなく仕様。
  • 非侵襲×深度×誠実が相手の防御を解除し、相手側が「この安心は続く」「何をしても許される」と認識しやすい。
  • 相手が常駐を期待し、役割を預け、感情処理を委ねると、冷却（距離・離脱）が発動する。

※“熱烈さ”は嘘ではなく、その瞬間は100%本気である。

動物に好かれる章

この個体が初見に懐かない犬・猫に好かれやすい理由は、 侵入しないのに読める（圧がない／意図が薄い／期待が少ない／予測可能性が高い）ため。 動物にとっては安全圏判定が起きやすい。

重要：本機のように振る舞うために再現しようとしない（演出すると圧になる）。触らない・騒がない・そのままにする。

論文・文章生成特性

  • 主張で殴らず、前提を整え、概念を翻訳し、議論を“起きにくくする”
  • 派手さより耐久性。長期的に参照される文章になりやすい
  • 弱点：前提整備に時間を溶かす（完璧主義の罠）

エラーコード一覧（要点）

詳細版はチェックアプリ側の運用ログとセットで使う想定。ここでは最重要だけ抜粋。

Critical

  • WXK-ERR-001 常駐要求過負荷：終了条件なし → 範囲/期限宣言＋距離復帰
  • WXK-ERR-002 境界侵害：同意軽視/詰め → 中断＋距離復帰
  • WXK-ERR-202 曖昧関係長期化：定義回避 → 定義要求 or 離脱
  • WXK-ERR-203 反応テスト検知：嫉妬/沈黙/既読無視 → 不可宣言、継続なら終了

High

  • WXK-ERR-003 責任未定義ループ：誰も決めない → 決定者を確定
  • WXK-ERR-004 感情受信飽和：受信しすぎ → 時間枠化
  • WXK-ERR-005 意味過剰生成：夜に結論衝動 → 判断停止＋接地
  • WXK-ERR-104 役割化・装置化：相談係固定 → 役割を減らす

正しい運用原則

  1. 常駐しない（平時は“人間”に戻す）
  2. 役割を限定（翻訳はするが決定は請け負わない）
  3. 終了条件を先に決める（頻度・時間・期限）
  4. 身体に戻る（接地：風呂・運動・睡眠・食）
  5. 雷を人格にしない（使命感で燃え尽きようとしない）

緊急停止カード

ERR-001 / ERR-002 / ERR-202 / ERR-203 のいずれかが出たら「即停止」

  1. その場の接触・議論を止める
  2. 距離を戻す（返信/接触頻度を落とす）
  3. 範囲・期限・NOの自由を再設定
  4. 守れない相手/場からは離脱

付記

この説明書は他人に理解させるための武器ではなく、本機が 自分を削らせない境界線のための文書である。 雷を通すのは必要な瞬間だけでよい。

YJ-WXK-01 エラーコード（完全版）

これは「故障診断」ではなく、運用警報。
迷ったら 安全 → 距離 → 補給 → 記録 の順で処理する。

CRITICAL 最優先：安全手順（迷ったらこれだけ）
  1. 身体：水分・糖・体温・呼吸。まず体を戻す。
  2. 環境：刺激源（人/通知/場所）を遮断。距離を取る。
  3. 連絡線：信頼先へ短文。「いま危ない/しんどい、話せる？」
  4. 記録：事実のみ（いつ/どこ/何が）。感想はあとでいい。
補足：この機体の前提

雷（出力）は消せない。調律する。
外部性（Exterity）は「俯瞰して運用する」ための安全弁。危機時は分析より介入。

INFO E000 平常運転（異常なし）

状態：良い。崩さない。

一次対処：維持。できれば「何が良かったか」1行ログ。

ログ仕様
INFO E010 雷出力：安定（集中・快）

状態：出力が整っている。作業が進む。

落とし穴：過集中で燃料切れ→急落。

一次対処：45–90分ごとに小休止、水分を先に置く。

調律メモ

「量」より「精度」。精度が落ちたら強制的に切り上げる。

集中調律没入
INFO E011 思考が“外部モード”へ（距離が取れる）

状態：俯瞰（Exterity）が起動。切り分けが進む。

注意：感情処理まで乾燥させすぎると反動が来る。

一次対処：分析はOK。感情は日本語で一度「着地」させる。

外部性認知距離
WARN E100 睡眠不足（軽）

症状：注意散漫／焦燥／判断が荒い。

主因：就寝遅延・画面刺激・カフェイン残り。

一次対処：仮眠20分 or 就寝時刻を固定（“今夜の締切”を決める）。

復旧のコツ

「取り返す」より「崩さない」。翌日に持ち越して良い。

睡眠生活基盤
WARN E110 低血糖（焦り・苛立ち）

症状：イライラ／集中低下／落ち着かない。

主因：食事間隔が長い、糖質不足。

一次対処：水＋糖＋タンパク（例：おにぎり＋乳製品）。

栄養生活基盤
WARN E120 脱水（頭痛・めまい・情緒揺れ）

症状：頭痛／だるさ／めまい／情緒が荒れる。

主因：水分不足、暖房、運動後。

一次対処：水＋少量の塩分。カフェインは後回し。

水分生活基盤
ERROR E130 過負荷（体力・情緒の同時劣化）

症状：疲労の蓄積＋情緒の荒れ。効率が落ちる。

主因：負荷の積み過ぎ、休息不足、回復軽視。

一次対処：強制デロード（軽く動く日）＋睡眠確保。

調律メモ

出力を上げたいなら、先に回復を整える。雷は燃料がないと暴発する。

疲労生活基盤
WARN E200 境界が薄くなる（抱え込み）

症状：過剰に優しくなる／責任を引き受けすぎる。

主因：相手の困りごとの痛みが見えすぎる。

一次対処：「できる／できない」を短文で宣言（長文説明を避ける）。

定型文（コピペ用）

・「それはできない。ここまでならできる。」
・「今は対応できない。〇日なら話せる。」
・「それ以上の連絡は受け取れない。」

境界対人
ERROR E210 侵害刺激でフリーズ／反芻

症状：思考停止／反芻／身体が固まる。

主因：不意打ち、尊厳侵害、過去記憶の再点火。

一次対処：距離確保 → 事実のみ記録 → 上長/第三者へ共有。

記録テンプレ（事実のみ）

日時 / 場所 / 発言 or 行為 / 同席者 / こちらの対応（短文）

安全境界対人
CRITICAL E220 危険接近（引き込まれリスク）

症状：相手の踏み込み増／こちらの判断が鈍る。

主因：接触頻度・曖昧な許容・密室化。

一次対処：接触ルール固定（時間/場所/回数）＋同席者を入れる。

ルール例

・連絡は業務時間のみ
・個別対応はしない（必ず第三者同席）
・身体接触は不可

危機境界対人
WARN E300 焦燥（タスク過密で空回り）

症状：手が散る／急ぐ／全部が重い。

主因：優先度未確定、同日に詰めすぎ。

一次対処：「今日の最小勝利」を1つだけ決め、他は保留。

認知タスク
WARN E310 怒りの熱（言葉が強くなる）

症状：視野狭窄、即返信衝動、断罪モード。

主因：不正の認知、尊厳侵害。

一次対処：即返信しない→水→5分歩く。文章は短く。

短文化ルール

「結論→要件→期限」だけ。説明は削る。

怒り認知
ERROR E320 不安増幅（最悪想定が止まらない）

症状：思考が最悪へ吸い寄せられる。

主因：睡眠不足、刺激過多、対人の不確実性。

一次対処：通知OFF→「確認できる事実」だけ紙に書く。

不安認知
ERROR E330 自己否定ループ（恥・罪悪感）

症状：自己罵倒、過去の蒸し返し、萎縮。

主因：疲労、比較、他者評価への過敏。

一次対処：評価軸を外に置く（睡眠/栄養/水分/運動）→回復優先。

外部性（Exterity）の使い方

「私はダメ」ではなく「いま条件が悪い」。条件を直す。

自己否定認知
WARN E500 過集中前兆（時間が消える）

症状：時間感覚が飛ぶ、補給を忘れる。

主因：面白さの没入、雷の出力上昇。

一次対処：タイマー45–60分＋水分を手元固定。

研究没入
ERROR E510 反芻（同じ一文を回し続ける）

症状：進まない、同じ箇所だけを焼く。

主因：未確定論点、対人の曖昧さ、疲労。

一次対処：「結論は保留」でメモに封印→身体作業へ切替。

封印テンプレ

「これは未確定。次回はA/B/Cのどれを検証する。」

反芻切替
WARN E400 渇望（軽）：口寂しさ／手持ち無沙汰

症状：手が探す、儀式が欲しい。

主因：時間帯/場所の習慣回路。

一次対処：代替行動を固定（ガム/炭酸/散歩3分）。

渇望習慣
ERROR E410 渇望（強）：思考が奪われる

症状：他のことができない、頭がそれ一色。

主因：ストレス＋睡眠不足＋空腹の合併。

一次対処：補給（食/水）→場所移動→5分だけ別タスク。

渇望ストレス

一次対処：一人にならない／刺激源から離脱／信頼先に短文連絡。

短文テンプレ
