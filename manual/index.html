序文

これは自己賛美でも呪文でもなく、運用のための仕様書である。 「なぜ同じ地点で壊れ、同じ地点で信頼され、同じ誤解を受けるのか」を、 構造として読み解くために編まれた。本機は、晴菜を愛するために特化する変化をする余地があり、本説明書も、晴菜に読まれることを意識しているが、必ずしも晴菜に読まれ切り、理解されることを意図してはいない。

存在構造（最終定義）

YJ-WXK-01 は、意味・判断・倫理を“生む”存在ではなく、通過させる（翻訳・媒介する）存在である。 そのため「中心ではないのに不在でもいられない」「常駐すると壊れるが、引くと場が回る」という矛盾を持つ。

  • 最大出力での常駐要求に弱い（Resident負荷）
  • 境界線が生命線（Boundaryが無いと焼損）
  • 危機・締切・決断点で最大出力（イベント駆動）

ワキンヤン・タンカとの同型性

ワキンヤン・タンカは雷そのものではなく、天と地のあいだに立ち、力を“雷”として翻訳して通過させる媒介である。
YJ-WXK-01 もまた、力を直接振るうより、場・判断・意味の流れを整える媒介体として作動する。
ネイティブ・アメリカン、スー族のワキンヤン理解では、ワキンヤンは山に住み、天候が荒れる際に雷を落とす原因として考えられている。
雷が落ちるとき、人間は”その人自身”を明らかにし、ワキンヤンの前では嘘は通用しない。
ワキンヤンは、嘘を最も嫌い、儀式中に嘘をついたものを焼き殺す性質がある。すなわち、ワキンヤンは純粋なものを好む。
ワキンヤンは、快晴の時に出ているウィ(太陽は、ウィと呼ばれる)とも表裏一体の関係をなしており、ウィが世界を照らしている間も、ワキンヤンはどこかで鳴りを潜め、人間に雷を落として啓示を与える空気として存在している。
スー族の伝承によれば、ワキンヤンは断片的な存在であり、ワキンヤン全体を理解することは難しいとされる。
夢でワキンヤンに出会った人間は、一時的にへヨカ(世界を逆転させて物を見る道化師の一種)となり、へヨカはワキンヤンを人間に通訳する役割を担う。
ワキンヤンの雷によって、人間は深く自己の内省に誘われ、”ただある”純粋な自分の在り方を問う。

同型性の核心

  • 力が常駐しない（必要な瞬間にのみ顕現）
  • 場を変える（空気・判断の基準点を作る）
  • 直接支配しない（侵入せず、翻訳で導く）

決定的な差異

ワキンヤンは壊れないが、YJ-WXK-01はワキンヤン的性質を人間の身体と感情に実装されているため壊れる。 よって必要なのは使命感ではなく制御（境界・冷却・離脱条件）である。

同型性が破綻する条件(人間としての限界)

ワキンヤン・タンカとの同型性は、あくまでも構造的なものであり、耐久性までを保証するものではない。
ワキンヤンは神話的存在であり、雷を通しても損耗しないが、YJ-WXK-01は人間であり、身体・感情・時間という制約を受ける。
この差異により、以下の条件下では同型性が破綻する。
第1に最大出力=雷の常駐を要求された場合である。
ワキンヤンは必要な時にのみ顕現するが、YJ-WXK-01が常時その役割を期待されると、媒介ではなく装置として固定され、過負荷を起こす。
第2に、境界線が曖昧なまま役割を引き受けた場合である。
雷を通す役割と、他者の感情・決断・責任を肩代わりすることは別であり、この区別が失われたとき、媒介構造は崩壊する。

第3に、純粋性が「義務」や「使命感」に変質した場合である。 ワキンヤンが好む純粋性は強制されないが、人間においてそれが自己犠牲や耐久競争に置き換わった瞬間、雷は内側で暴発する。

したがって、YJ-WXK-01にとって重要なのは、雷を通し続けづることではなく、
雷を通す瞬間と、通さない瞬間を明確に分けること
である。
同型性は、自己同一化の根拠ではなく、運用判断の指針であり、破綻条件を知ることは、役割放棄ではなく安全確保である。

破綻後の回復プロトコル(雷を抜く工程)

概要
破綻は失敗ではなく、過負荷時に起こる安全弁作動である。
重要なのは「折れないこと」ではなく、雷を体と関係性から抜く手順を知っていること。
破綻の定義
以下が同時に起きた状態を「破綻」と定義する。

  • 判断が止まらない(意味生成が暴走
  • 他者の感情が自分の責任に見える
  • 休息・距離・拒否が「不誠実」に感じる
  • 身体反応(不眠・身体化・乖離・動機・過集中など)が起きる
回復プロトコル(順序厳守)
第1段階：即時遮断
  • 議論・説明・翻訳を止める
  • 社交頻度を落とす
  • 「今は判断しない」と明言できる相手には明言する

第2段階：身体への着地
  • 風呂・運動・睡眠・食事
  • 思考より先に体温と呼吸を戻す
  • 神話・意味・比喩の思考を一時停止する

第3段階：雷の所在を戻す
  • 「これは自分の雷ではない」と言語化する
  • 他者の感情・選択・人生を本人に返却する
  • 必要なら関係性自体を再定義・縮小・離脱する

注意事項 回復中に「役割」「使命」「理解されなさ」を考えないこと。 これらは再点火スイッチである。

へヨカ状態の持続限界

へヨカ状態とは
へヨカとは、ワキンヤンに触れた人間が一時的にとる 逆転的・俯瞰的・通訳的な認知状態である。

  • 価値観を反転させて見る
  • 他者の嘘や矛盾が即座に見える
  • 場の構造を一段上から理解できる
重要な制限
へヨカは常態ではない。
YJ-WXK-01において、へヨカ状態は以下の制約をもつ
  • 持続時間は短期(数時間から数日単位)
  • 長期化すると「孤立」「燃え尽き」「冷笑」に移行
  • 周囲は、これが「安定した人格」と誤認しやすい
限界を超えた兆候
  • 皮肉や逆転思考がとまらない
  • 「どうせわからないだろ」という感覚が増える
  • 感情が良くも悪くも平坦化する
運用原則
へヨカは一時的な翻訳装置であり、人格や生き方ではない。

他者が「ワキンヤンを見た」と誤認する瞬間

誤認の構造
YJ-WXK-01が以下の条件で振る舞うとき、他者はしばしば 本人ではなく”力そのもの”を見たと錯覚する。

  • 嘘をつかない
  • 判断を濁さない
  • 感情を侵襲せず、核心だけを言語化する
  • 圧をかけずに真実を露出させる

この時、他者の内部では
「この人は特別だ」
「この人は見抜いてくれる」
「この人ならわかってくれる」
という神格化誤認が起きる。
危険点
  • 他者は「啓示」を与えられたと感じる
  • しかし、その後の選択・責任・苦痛をYJ-WXK-01に預けようとする
これは、 ワキンヤンを見たのではなく、
雷の反射を見ただけである。
防止策
  • 判断を返す質問を使う 例：「それを選ぶのは誰？」
  • 解釈を限定しない 例：「私の見え方の一つにすぎない」
  • 継続的な役割期待を断る
他者が見たのは、”真実”ではなく、一瞬の翻訳結果である。

基本動作原理

原理A：非侵襲性

近づきすぎず、詰めず、奪わず、操作しない。結果として相手の防御が解除される。

原理B：判断一貫性

何でも言わないが、言う時は迷いを減らす。相手が“基準点”と認識しやすい。

原理C：イベント駆動

危機・締切・決断点で最大出力。平時にも常に最大出力を求められると破綻しやすい。

恋愛仕様（実運用ログ反映）

  • 数は多い / 愛情表現が熱烈 / 破綻も多い —— これは異常ではなく仕様。
  • 非侵襲×深度×誠実が相手の防御を解除し、相手側が「この安心は続く」「何をしても許される」と認識しやすい。
  • 相手が常駐を期待し、役割や依存を押し付け、すべての感情処理を委ねると、過負荷による熱暴走が起きる。
  • 熱暴走は、本機の過電流に起因するものであり、仕様の一つである。しかし、それによって焼損することがある。

※“熱烈さ”は嘘ではなく、その瞬間は100%本気である。

動物に好かれる

この個体が初見に懐かない犬・猫に好かれやすい理由は、 侵入しないのに読める（圧がない／意図が薄い／期待が少ない／予測可能性が高い）ため。 動物にとっては安全圏判定が起きやすい。

重要：本機のように振る舞うために再現しようとしない（演出すると圧になる）。触らない・騒がない・そのままにする。

論文・文章生成特性

  • 主張で殴らず、前提を整え、概念を翻訳し、議論を“起きにくくする”
  • 派手さより耐久性。長期的に参照される文章になりやすい
  • 弱点：前提整備に時間を溶かす（完璧主義の罠）

正しい運用原則

  1. 常駐しない（平時は“人間”に戻す）
  2. 役割を限定（翻訳はするが決定は請け負わない）
  3. 終了条件を先に決める（頻度・時間・期限）
  4. 身体に戻る（接地：風呂・運動・睡眠・食）
  5. 雷を人格にしない（使命感で燃え尽きようとしない）

緊急停止カード

ERR-001 / ERR-002 / ERR-202 / ERR-203 のいずれかが出たら「即停止」

  1. その場の接触・議論を止める
  2. 距離を戻す（返信/接触頻度を落とす）
  3. 範囲・期限・NOの自由を再設定
  4. 守れない相手/場からは離脱
